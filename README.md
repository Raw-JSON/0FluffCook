* **The Goal:** Scrape recipes directly from websites to prevent AI hallucinations.
* **The Reality:** Many websites block client-side scrapers (CORS). We use proxies (`corsproxy.io` and `allorigins.win`), but heavily secured sites might still fail to load.
* **Fallback:** If the site blocks the scraper, you can paste the recipe text manually, and the AI will clean it perfectly.

## üõ† Features
* **Multi-Proxy Scraper:** Attempts to bypass CORS blocks to read real website HTML.
* **Smart Cleaning:** AI removes repetitive text (e.g., "For the Sauce:") automatically.
* **Local Storage:** Recipes are saved to your browser cache. Zero servers.
* **Favorites:** Pin your top meals.
* **Import/Export:** Backup your database to JSON.
